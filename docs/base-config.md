---
`Mkbug.js`用于解析配置文件`*.conf`的工具类。

## 方法 Config()
配置管理对象的构造函数
### 参数
| 参数 | 说明 | 类型 | 必填 | 默认值 |
| ---- | ---- | ---- | ---- | ---- |
| `name` | 配置名，`*。conf`文件名 | `String` | 必填 | 无 |
| `opts` | 初始化参数，编码格式 | `Object` | 非必填 | `{ encoding: 'utf8', path: './src' }` |
### 返回
`Object`: `*.conf`文件映射的键值对对象

## 方法 super()
父类的构造函数，如果派生新的基类必须执行。
### 参数
| 参数 | 说明 | 类型 | 必填 | 默认值 |
| ---- | ---- | ---- | ---- | ---- |
| 无 | 无 | 无 | 无 | 无 |
### 返回
无
